# Mazegohan Sushi & Poke 🍣

Página estática para direcionar clientes direto para a loja no iFood.

## Como usar

1. **Adicione a imagem de fundo**
   - Coloque o arquivo `fundo.jpg` na raiz do repositório (mesmo nível do `index.html`)
   - A imagem deve ter resolução mínima de 828x600px para melhor qualidade

2. **Faça o deploy**
   - Ative GitHub Pages nas configurações do repositório
   - Selecione a branch `main` como fonte
   - Sua página estará disponível em: `https://o-melhor-delivery.github.io/sushi-e-poke/`

## Características

✅ Botão posicionado dentro do campo branco da imagem  
✅ Link direto para iFood (sem problemas de navegação)  
✅ Responsivo para mobile e desktop  
✅ Design simples e rápido
